## Interactive States

### Default:
- All buttons enabled
- X button visible
- Backdrop clickable

### Loading (After Confirmation):
- Logout button shows spinner
- All buttons disabled
- X button hidden
- Backdrop not clickable
- "Logging out..." text

### Success:
- Modal closes
- Redirect to home page
- User logged out

### Error:
- Loading stops
- Buttons re-enabled
- X button visible
- Backdrop clickable
- Error logged to console
